Group project name: Smart green office plus
智慧綠色辦公室影像辨識系統 — 專案書面介紹

一、專案背景與動機
隨著企業辦公空間逐漸數位化，能源浪費、安全風險與工作效率問題卻仍普遍存在。例如：空間無人使用但設備仍持續運作、飲料及食物靠近電子設備造成潛在風險、桌面雜亂影響專注力，卻缺乏即時提醒。
多數辦公室仍依賴人工管理，缺乏即時感知空間狀態的能力。
因此我們提出Smart Green Office Plus，透過影像辨識與即時分析，讓辦公空間具備「看見、理解並給予建議」的能力，達到節能、安全與效率提升的目標。

二、系統整體架構
本系統由以下三個核心技術組成：
1.硬體設備:
Raspberry Pi
PiCamera2 攝影機

2.影像辨識核心:
YOLOv8 物件偵測模型
即時辨識影像中的人與物件
智慧分析與視覺化

3.OpenCV 即時畫面標註:
指標計算與提示生成模組

系統運作流程如下：
攝影機擷取畫面 → YOLO 辨識物件 → 物件分類與統計 → 高階指標推估 → 即時提示顯示

三、主要偵測對象說明

系統基於 YOLOv8 的 COCO 類別，將偵測物件分為以下幾類：

（一）人員偵測
person:用於判斷空間是否有人、使用密度與狀態推估

（二）電子設備偵測（Devices）
tv,laptop,cell phone,keyboard,mouse:用於分析設備使用情況與估算能源消耗

（三）飲料類偵測（Drinks）
bottle,cup,wine glass:用於補水提醒與飲料靠近設備的安全風險分析

（四）食物類偵測（Food）
banana, apple, sandwich, pizza, donut 等:判斷食物是否靠近電子設備，提示清潔或安全風險

（五）桌面雜物偵測（Clutter）
book,remote,backpack,keyboard,mouse,scissors 等:用於評估桌面整潔程度與工作環境品質

（六）潛在危險物品偵測（Safety）
knife,scissors:用於即時安全提醒，降低意外發生風險

四、系統功能模組詳細說明
1️.人數與空間使用狀態分析:即時計算空間內人數，判斷空間是否處於使用中或閒置狀態

2️.電子設備使用量分析:統計所有偵測到的設備數量，作為能源消耗與節能判斷依據

3️.會議模式（Meeting Mode）判斷:當系統同時偵測到多位人員、多個顯示設備，則自動判斷為「會議模式」，可作為未來自動切換燈光、錄音或節能策略的依據。
4️.桌面整潔度（Clutter Index）分析:依雜物數量推估桌面整潔程度，雜物越多，代表環境干擾越高
5️.專注度指標（Focus Score）:系統綜合以下因素計算專注度（0–100）:
  (1)人數多寡
  (2)桌面雜物數量
  (3)是否存在飲料靠近設備等干擾風險
進而提供對工作環境品質的即時評估
6️.節能指標（Green Score）根據人員數量、電子設備數量推估目前空間的能源使用效率，數值越高代表越節能。
7️.估計耗電量（Estimated Power）:透過假設功耗模型(筆電、螢幕、手機等設備)，推算目前空間的相對耗電狀態，重點在於比較趨勢而非精準數值。
8️.飲料靠近設備風險分析:計算飲料與筆電之間的距離，若距離過近，觸發風險提醒
9️.安全風險即時提示:偵測到刀具或剪刀、食物或飲料靠近電子設備，系統會即時顯示警示文字，提醒使用者注意安全。
10.智慧建議（Tips）生成:系統會根據當前狀態，自動產生建議，例如：
  (1)空間無人 → 建議關閉設備
  (2)有人但無飲料 → 補水提醒
  (3)桌面雜亂 → 提醒整理
  (4)會議模式 → 建議切換節能設定

五、系統設計理念與特色
1.非監控導向：不記錄個人身分，只分析空間狀態
2.即時回饋：讓使用者立即理解環境狀況
3.高階資訊轉換：從物件偵測提升到行為與狀態判斷
4.可擴充性高：可整合燈光、空調、ESG 系統

六、應用情境與未來發展
本系統可應用於：
1.智慧辦公室
2.校園自習空間
3.企業節能管理
4.智慧建築與 ESG 評估

未來可進一步擴充：

七、結論
Smart Green Office Plus不僅是一個影像辨識系統，更是一個能理解辦公空間狀態、主動提供建議的智慧助理。
透過 AI 技術，讓空間不再只是被動使用，而是成為 節能、安全且友善的智慧環境。
